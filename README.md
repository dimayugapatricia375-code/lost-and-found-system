# Lost and Found Management System

A web-based Lost and Found Management System built using HTML, CSS, JavaScript, PHP, and MySQL.

## Features
- User registration
- Password length checker (JavaScript)
- Report lost and found items
- CRUD operations
- MySQL database integration

## Technologies Used
- HTML
- CSS
- JavaScript
- PHP

## How to Run
1. Import `database.sql` into phpMyAdmin.
2. Place files inside the htdocs folder (XAMPP).
3. Open browser and go to localhost.
